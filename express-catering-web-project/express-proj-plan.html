EXPRESS Catering Services

Project Description

This project will create a website for EXPRESS Catering Services, offering detailed menus, customizable event packages, and an easy booking system for clients. The site will feature secure payment processing and an inquiry section to streamline the catering ordering process.

Project Reason

The website will enhance the business visibility and provide a convenient way for clients to explore options and book services online, expanding the business reach and attracting more customers.

Project Purpose

The website will make it easier for clients to browse catering options, request quotes, and book services. It will also act as a marketing tool, showcasing the service offerings and building trust, ultimately driving growth and improving customer engagement.

Website Structure

The website will have a total of three pages:

  • Home Page: Overview of the catering service, highlights of featured offerings, and a call-to-action for booking.
  • Menu & Packages Page: Detailed listing of menu items and customizable event packages.
  • Contact & Booking Page: Inquiry form, booking options, and secure payment processing details.

Color Schema

The website will use the following colors:

  • #d35400 (Orange): Used for headings and accent elements to add warmth and energy.
  • #2c3e50 (Dark Blue): Used for the header and navigation bar to provide contrast and elegance.
  • #f8f8f8 (Light Gray): Used for the background to ensure readability and a clean look.

Typography

The website will use the following fonts:

  • Oxygen, sans-serif: Used for body text to ensure readability and simplicity.
  • Roboto, serif: Used for headings to create a formal and elegant feel.

Wireframe

The wireframe will consist of two layouts:

  • Mobile View: A simplified, stacked layout with easy navigation.
  • Desktop View: A wider, structured layout with sections for featured offerings, testimonials, and an easy-to-find booking button.
